## download
```
    git clone https://github.com/VTramon/web_nlw_heat.git
    
    cd web_nlw_heat
```

---

## iniciando peojeto

#### Caso queira rodar em sua propria maquina:

```
    yarn 
    yarn dev
    
    ## OU ##
    
    npm install
    npm run dev
```
#### Caso queira roda com o docker:

```
    docker build .
    docker run <IMAGE ID>
```
